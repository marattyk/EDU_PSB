# EDU_PSB
Образовательная платформа для ПСБ
Шаги по запуску:

1) Скачать файл EDU.7z, либо в EDU_PSB/EDU.7z нажать "view raw"
2) Распаковать архив в удобной папке
3) Создать новый проект -> вставить в него папку EDU
4) Установить необходимые библиотеки
pip install fastapi==0.104.1
pip install uvicorn==0.24.0
pip install pydantic==2.5.0
pip install python-multipart==0.0.6
pip install fastapi uvicorn sqlite3
5) Запустить файл EDU/documentation/main.py
6) Открыть в браузере http://localhost:8000/ (в опере может не открываться, желательно использовать Microsoft Edge или Яндекс браузер)
7) Логины и пароли для входа указаны под строкой ввода пароля
